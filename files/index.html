Índice de Ejercicios de JavaScript

Haz clic en el título de un ejercicio para abrirlo en una nueva pestaña.

  • Ejercicio 1: Validación de Formulario

    Un formulario simple que valida los campos del usuario usando `alert` para mostrar errores.

  • Ejercicio 2: sessionStorage

    Demuestra cómo guardar datos que solo persisten durante la sesión de una pestaña del navegador.

  • Ejercicio 3: localStorage (Bloc de Notas)

    Un bloc de notas que guarda el texto de forma permanente en el navegador, incluso si se cierra y se vuelve a abrir.

  • Ejercicio 4: Cookies

    Recuerda el nombre de un usuario entre visitas utilizando una cookie para ofrecer un saludo personalizado.
